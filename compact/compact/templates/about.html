About Compact Input/Output Report Generator
This impact analysis tool was designed to calculate the effects of state-level industry spending on other industries in the same area.

The initial data set for this project is obtained from the Bureau of Labor Statistics API. This data is then parsed and run through an impact analysis model designed to calculate the direct, indirect, and induced impacts of a given industry (identified by NAICS code) across three categories: Revenue, Employment, and Wages.

Data is then rendered similar to a Direct Impact table for a national-scale impact analysis report. All pages are printer friendly and allow for individual export of tables and charts.

Please note that for industries in which employment data is not disclosed by by the Bureau of Labor Statistics, calculations were provided based on estimates.
